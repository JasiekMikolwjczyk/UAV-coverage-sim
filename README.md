# uav-coverage-sim

Recreated skeleton. You must provide:
- PX4 source at `submodules/PX4-Autopilot` (git submodule or checkout).
- Gazebo models for x500/x500_base (if not in PX4 path).

Scripts:
- `start.sh` (single UAV)
- `start_multi.sh` (3 UAVs)
- `run_mission.sh` (single mission + coverage)
- `run_multi_mission.sh` (multi mission + coverage score)

ROS2 workspace in `ros2_ws/`.
